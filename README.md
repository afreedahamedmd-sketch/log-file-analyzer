# Log File Analyzer

A simple Python project that analyzes system log files and summarizes log levels such as INFO, WARNING, and ERROR.

## Project Overview
This tool reads a log file and counts how many INFO, WARNING, and ERROR messages occur in the file. It helps understand system activity and detect issues quickly.

## Features
- Reads log files
- Detects INFO, WARNING, and ERROR messages
- Generates a summary report
